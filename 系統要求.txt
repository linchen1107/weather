我想要首頁顯示當前所在地的天氣資訊、台灣的縣市圖(鼠標移到該縣市就會顯示當天的天氣資訊)，
第二頁顯示一周的天氣預報的卡片以及台灣的縣市圖，
第三頁標籤則是一周天氣預報的圖表(例如折線圖)以及台灣的縣市圖。
要這麼多縣市圖的主要原因是可以讓使用者選擇要查看哪個地方的天氣資訊，也可以額外加一個搜尋功能。「我的位置」按鈕置於導覽列的最右方。

前端：給每個 <path> 綁 mouseover / mouseout 事件，並在 Tooltip 顯示天氣。
後端：把中央氣象局 / OpenWeather 的資料整理成 JSON，前端 AJAX 拿到後，依縣市名稱對應。




準備台灣地圖資料：

你可以找一份台灣各縣市的 SVG 地圖檔，每個縣市用單獨的 <path>（或其他 SVG 元素）標示，並賦予對應的 id（例如台北市用 id="Taipei"，高雄市用 id="Kaohsiung" 等）。
嵌入地圖到網頁：

將這份 SVG 地圖嵌入到你的 HTML 當中（可以直接在模板內嵌或利用 <object>/<embed> 標籤）。
綁定滑鼠事件：

用 JavaScript 為每個縣市的區域綁定 mouseover 與 mouseout 事件。
當滑鼠移到某個縣市時（例如觸發 mouseover），利用預先取得或從後端請求該縣市的天氣資料，並用 Tooltip、彈跳視窗或浮動框顯示出來。
顯示與隱藏天氣資訊：

當滑鼠移入時，計算 Tooltip 顯示的位置（可以跟著鼠標），並填入該縣市的天氣摘要。
當滑鼠移出時，隱藏 Tooltip。
何時實現：

這個功能可在前端頁面初始化後立即進行（即在頁面 load 完成後就載入地圖並初始化事件），也可以根據用戶操作動態加載。
如果你的天氣資料已經有各縣市的預報（例如從中央氣象局 API 中取得），那麼你可以將所有縣市的預報資料提前載入，並在事件中直接查詢對應縣市的資料來顯示。
